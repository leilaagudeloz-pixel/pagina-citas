# Wireframe Formulario Registro de Cita Médica

## Descripción
Formulario de registro de cita médica para un sistema fisioterapéutico.
Permite registrar datos del paciente, citas previas, estado de salud,
actividad realizada en la cita y fórmula médica.

## Estructura del repositorio
- `index.html` → formulario con semántica HTML aplicada
- `css/color.css` → estilos del formulario
- `wireframe.pdf` → diseño previo del formulario en Figma

## Tecnologías usadas
- HTML5 semántico
- CSS3

## Etiquetas semánticas usadas
- `<header>` → encabezado de la página
- `<main>` → contenido principal
- `<section>` → cada sección del formulario
- `<form>` → formulario de registro
- `<footer>` → botones de acción
- `<label>` → etiquetas de cada campo
- `<input>` → campos de texto, fecha, número y teléfono
- `<textarea>` → campos de texto largo
